Use Type: Uses identified with a (I) are considered permitted uses; if meeting all applicable standards, such uses may be processed as a zoning certificate application through the Type I procedure. Uses identified with a (II) or (III) are considered conditional uses which must first obtain approval through the conditional use permit process utilizing the Type II or III procedure respectively. Approved conditional uses must obtain zoning certificate approval prior to the commencement of any development activity as required by Paragraph 62.111. Uses identified with a VAR (Variable) are classified as either permitted uses or conditional uses within the detailed regulations of this chapter.
